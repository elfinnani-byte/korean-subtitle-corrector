# 번역가 교육자료 요약 (바른번역미디어)

이 문서는 사용자가 공유한 번역가 교육자료(PDF 4종 + 구글 문서 1종)의 핵심 내용을
이 프로젝트(korean-subtitle-corrector)의 관점에서 정리한 것이다. 원문을 그대로
옮긴 게 아니라, 이 도구의 개발에 참고할 수 있도록 규칙 위주로 재정리했다.

  `맞춤법 심화 노트.pdf`, `외표법 입문 노트.pdf`, `외표법 심화 노트.pdf`
- 구글 문서: https://docs.google.com/document/d/1_g01euEErahy-mSrFrLV2WjcaJz206DCWJh6I-OTlGs
  ("우리말 표기 원칙" — 위 PDF보다 훨씬 포괄적인 통합 문서)

각 항목 뒤에 이 프로젝트에서의 현재 상태를 표시했다: **[구현됨]** / **[범위 밖]**
(PRD.md에 이미 범위 밖으로 명시됨) / **[백로그 후보]**(아직 코드로 옮기지 않았지만
가능성 있음).

## 1. 띄어쓰기 기본 원칙

- 체언·용언·수식언(관형사/부사)은 띄어 씀이 원칙, 조사·접두사·접미사·어간·어미는
  붙임 — **[구현됨]** (`_ATTACH_TAGS`/`_MANDATORY_BOUNDARY_TAGS`, 제41항)
- 판별 순서: 사전에 통째로 검색 → 접사 여부 확인 → 고유명사/전문용어 여부 확인 →
  그래도 안 되면 못 붙임 — **[구현됨]** (`word_exists`/`compound_status` 우선순위)
- 고유명사·전문용어는 원칙은 띄어쓰기, 붙이려면 전부 붙임(부분만 붙이는 것 불가) —
  **[구현됨]** (`_TERM_COMPOUND_TAGS`, §21)

## 2. 의존명사·수 표기

- 단위성 의존명사: 순서/차례를 뜻하거나 아라비아 숫자 뒤에 오면 붙임 허용
  (일층/1층 등) — **[백로그 후보]** — 지금은 "20만" 같은 순수 숫자 표현만
  다룸(제44항), "일층" 류의 순서성 의존명사는 아직 코드 없음
- "제-"(접두사)는 항상 붙임 — **[구현됨]** (`_TERM_COMPOUND_TAGS`의 XPN)
- "대"(VS. 뜻, 3 대 1): 항상 띄움, 예외 "일대일" — **[백로그 후보]**
- 숫자는 우리말 숫자+우리말 단위, 한자어 숫자+한자어 단위로 어울려야 함,
  숫자 범위 표기("2천-3천 명"), 만 단위 이내 붙임(제44항) —
  **[일부 구현됨]** (20만류만 §21에서 처리, 나머지는 백로그)

## 3. 차별적 표현 → 순화어

간질→뇌전증, 벙어리장갑→손모아장갑, 반팔→반소매 등 — **[구현됨]**
(`common_errors.py`의 DISCRIMINATORY_TERMS/PURIFIED_TERMS, 반팔은 순화어라
자동교정 아닌 플래그로 분류)

## 4. 고유명사 띄어쓰기 (지명/거리/음식명)

외국 지명은 붙임, "대로/로/가"는 붙임이지만 "길/거리"는 원칙적으로 띄되 고유명사로
보아 붙임 허용, 요리/음료명은 붙이되 식재료명은 띄움 — **[범위 밖]**(§13에서 이미
"고유명사 외래어 표기는 항상 확인 플래그"로 정책 확정, 식재료/요리 구분까지는
아직 미세분류 — 백로그 후보)

## 5. 외래어 표기법 (외표법 노트 전체)

IPA 대조표 기반 세부 규칙(어말 파열음 받침/으 표기, [oʊ]→오, -ton→턴 등),
인명/지명 원지음 원칙, 중국(1911년 신해혁명 기준 한자어/중국어 구분), 일본(항상
일본어 발음), 상표명(한국 지사 등록상표 우선), 형용사형 변환(Persian→페르시아) —
**[범위 밖]** PRD §5에 이미 명시: "이 규칙 엔진은 영어 하나만 다루는 것보다 훨씬
큰 작업, v2 이후 백로그". 지금 엔진은 kornorms(국립국어원 외래어 용례) API
조회로만 처리하고, 세부 표기 규칙을 직접 구현하지 않음.

## 6. 문장 부호

쉼표(감탄사 뒤 필수, "그러나/또한/사실" 뒤에는 안 붙임), 말줄임표(현재 점 3개
허용, 영상자막은 뒤 마침표 생략), 물결표 대신 붙임표 사용 권장, 따옴표도
번역 필요, 세미콜론 용법 — **[범위 밖]** (PRD §9에 "자막 고유의 문장부호 관례는
별개"로 이미 명시, 문장부호는 이 엔진이 아예 다루지 않는 영역)

## 7. 단위 표기

미터법 변환 원칙(항공/미식축구/해양은 예외), km/m 함부로 변환 금지, 자동차 크기
mm+전장/전폭/전고, 톤(ton) 국가별 차이, 신장 환산, 과학 기호 금지(㎡→제곱미터),
컴퓨터 단위(GB/MB/KB) — **[범위 밖]** (단위 변환은 번역 내용 검증 영역이라 이
맞춤법 교정기의 대상이 아님)

## 8. 맞춤법 세부 규칙 (이 엔진과 가장 밀접)

- 명사 뒤 접사처럼 쓰이는 경우("오렌지잎")는 명사구 뒤엔 못 붙임 —
  **[백로그 후보]**
- 감탄사 뒤 쉼표("뭐", "거봐") — **[범위 밖]**(문장부호)
- 불규칙 활용형(누렇다→누레/누러니/누렇소) — **[구현됨]**
  (§21에서 `.startswith("VV")`로 불규칙 활용 어간 태그 대응)
- "이따"(단독) vs "있다"(좀 있다 봐) — **[백로그 후보]**
- 종결어미 "-요/이요", "-(ㄴ)대/-(ㄴ)데", "-재"(그러재), "-던/든" —
  **[일부 구현됨]** ("요" 하나는 §17/§20에서 처리, 나머지 종결어미 구분은
  백로그)
- **동사/접사 구분법(시키다/당하다/하다/받다/되다/드리다)**: 동작성 없는
  명사 뒤엔 동사(띄움), 동작성 있는 명사 뒤엔 접사(붙임), 동작성 없어도
  피동성 있으면 접사(붙임) — **[구현됨]** (§21/§22, `_is_action_noun`,
  `_PASSIVE_ONLY_BATDA_NOUNS` — 지금은 "받다"만 구현, 시키다/당하다/하다/되다/
  드리다는 아직 이 일반화된 판별 로직을 안 씀 — **백로그 후보**: 같은 방식을
  나머지 5개 동사에도 확장)
- 보조 용언(늙어 가다/늙어가다 등 붙임 허용, 합성어+보조용언은 항상 붙임,
  "-어지다/-어하다"는 항상 붙임) — **[구현됨]** (§5 `correct_aux_verb_spacing`)
- 의존명사 "뿐" — **[백로그 후보]** (제42항 잔여 항목, §5에 이미 "다음 검증
  대상"으로 기록됨)
- "맨"(순純 한국어 접두사, 항상 붙임) — **[백로그 후보]**
- 수효 표기(한번/한 번, 한잔/한 잔, 한입/한 입 — 행위면 붙임, 수효면 띔) —
  **[범위 밖, 의도적]** PRD §5에 "판단 보류, 항상 확인 필요"로 이미 명시된
  대표적 애매 사례
- "뭐 하다"(의문) vs "뭐하다"(특정 용법) — **[백로그 후보]**
- "안 되다"(금지) vs "안되다"(안타깝다) — **[구현됨]** (§20 `_andoeda_forces_split`)
- "못 하다"(능력 전무/방해) vs "못하다"(능력 부족) — **[백로그 후보]** —
  "안되다/안 되다"와 완전히 같은 구조의 문제라 같은 방식(조건·전제 어미 등
  문맥 신호)으로 확장 가능해 보임
- "잘못하다"/"잘 못하다"/"잘 안 하다", "잘 하다"/"잘되다"/"잘 되다" —
  **[백로그 후보]** — 위와 같은 계열
- "N 같다"("처럼"으로 치환 가능하면 붙임), "N+만 하다" — **[백로그 후보]**
- 의미가 달라지는 띄어쓰기(집어던지다/집어 던지다, 뱃속/집안/다음날/잘살다 등
  관용구성 복합어) — **[일부 구현됨]** (뱃속/입안류 맞춤법 개정 사항은 §5의
  compound_status 사전 조회로 자동 커버됨, 나머지는 개별 확인 필요)

## 9. 번역 스타일/표현 (영문법 번역, 번역투, 슬랭, 전문용어, 관용구, 게임 용어 등)

**[범위 밖]** PRD §13에 이미 명시: "번역투|오역|윤문 섹션은 아예 성격이 다른
카테고리 — 규정상 맞다/틀리다가 아니라 번역 스타일 판단이라 이 도구의 핵심
원칙(권위 있는 규범 근거 기반)으로 검증할 방법이 없다." 전문용어(의료/경찰/
군대/건축) 사전도 §9에 "도메인별 전문용어 사전"으로 이미 v2 이후 백로그로
분류됨.

## 10. 체크리스트 (자가 점검용, 번역가가 직접 확인하는 항목)

시키다/당하다/하다/받다/되다/드리다, 뱃속/집안/다음날, 외래어 표기(알콜→알코올,
나트륨→소듐), 자막 나이 표기(말자막 "-살" vs 화면자막 "-세"), 단위 범위 표기,
"&"→및/겸/또는, "넣을지 말지 헷갈리면 빼라"(의/것/게 되다/게 하다 등 군더더기
표현) — 대부분 위 항목들과 중복되거나 **[범위 밖]**(번역 스타일)에 해당.

## 정리: 이 문서에서 나온 새 백로그 후보 (우선순위 순)

1. "시키다/당하다/하다/되다/드리다"에도 "받다"와 같은 동작성/피동성 판별 로직
   확장 — 이미 검증된 패턴이라 상대적으로 착수하기 쉬움
2. "못 하다"/"못하다", "잘못하다"류 — "안되다/안 되다"와 동일 구조라 재사용 가능
3. 순서성 의존명사(제-, 층, 대) 확장
4. 의존명사 "뿐", "맨" 접두사 등 제42항 잔여 항목
